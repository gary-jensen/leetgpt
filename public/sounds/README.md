# Sound Effects Directory

This directory contains sound effects for the BitSchool application.

## Required Sound Files

Please download and add the following sound files to this directory:

### 1. success.mp3

-   **Purpose**: Played when code execution succeeds
-   **Duration**: 1-3 seconds recommended
-   **Style**: Positive, satisfying sound (e.g., chime, ding, success beep)
-   **Volume**: Will be set to 50% in the application

### 2. error.mp3

-   **Purpose**: Played when code execution fails
-   **Duration**: 1-3 seconds recommended
-   **Style**: Negative, alerting sound (e.g., error beep, warning tone)
-   **Volume**: Will be set to 50% in the application

### 3. levelup.mp3

-   **Purpose**: Played when user levels up
-   **Duration**: 2-4 seconds recommended
-   **Style**: Celebratory, achievement sound (e.g., fanfare, level up chime)
-   **Volume**: Will be set to 70% in the application

## Recommended Sources

-   [Freesound.org](https://freesound.org) - Free sound effects with Creative Commons licenses
-   [Zapsplat](https://zapsplat.com) - Professional sound effects (free account available)
-   [Mixkit](https://mixkit.co/free-sound-effects/) - Free sound effects
-   [OpenGameArt](https://opengameart.org) - Game-focused sound effects

## File Format

-   **Format**: MP3
-   **Quality**: 128kbps or higher recommended
-   **Sample Rate**: 44.1kHz recommended

## Notes

-   The application will gracefully handle missing sound files
-   Sounds are muted by default until user interaction occurs (browser autoplay policy)
-   Users can toggle sound on/off in the application
